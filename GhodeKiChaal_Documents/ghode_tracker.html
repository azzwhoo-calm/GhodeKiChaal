Every checkbox saves to progress.json in your repo, so your colleague sees it within ~20 s. Each of you does this once, on your own device.

  1. Create a private repo (or use the game repo). Commit this HTML file and an empty progress.json containing {"schemaVersion":1,"items":{}}.
  2. GitHub → Settings → Developer settings → Fine-grained tokens → Generate. Repository access: only that repo. Permissions: Contents → Read and write. Nothing else.
  3. Paste everything below. The token stays in this browser only (localStorage) — never share it or commit it.
